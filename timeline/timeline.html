My Timeline

Alongside this, I’m also learning web development. Currently, I’m enrolled in the "Advanced NLP for Beginners" course as part of the Winter in Data Science (WIDS) program.

    I deepened my knowledge of C++ through CS101 at IIT Bombay. Building on this foundation, I started solving problems on LeetCode and participated in entry-level Codeforces tournaments. Currently, I am learning Dynamic Programming.

    I participated in the XLR8 competition, where I built a gesture-controlled robot. This experience introduced me to Raspberry pi, Thonny IDE, and python.

    I explored game development using Processing software, creating simple yet engaging games like Flappy Bird and Ping Pong. These games were uniquely controlled using hardware I built myself, blending coding and electronics seamlessly.

    I delved into IoT development using the ESP8266 module, working with sensors like soil moisture, GPS, and inertial sensors to create smart projects. I also learned multi-layer PCB designing using DipTrace, enhancing my ability to create complex electronic designs.

  • 2019: Arduino

    I explored Arduino programming with its C++ variant, experimenting with sensors, modules, and shields to create fun and interactive projects. This experience deepened my understanding of embedded systems and hands-on electronics.

  • 2018: Intro to Electronics

    I’ve learned about key electronic components like capacitors, 555 Timer ICs, LM358 ICs, motor driver ICs, BC547 transistors, and IR/touch sensors. These basics have helped me understand and create simple electronic circuits.
